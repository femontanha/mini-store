## Setup

This project uses [Node](https://nodejs.org/en/) and [NPM](https://nodejs.org/en/). Install all dependencies running:

```
npm i
```

## Running Server

We use `GruntJS` as taskrunner to run our server and serve our assets.

```
grunt serve
```
